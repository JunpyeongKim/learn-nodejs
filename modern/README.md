# 모던 웹을 위한 Node.js 프로그래밍
- 윤인성, 한빛 미디어

## Installation
1. Mac
- www.nodejs.org --> Download the installer --> Install
 
2. Ubuntu
- www.nodejs.org --> Download the source code
- $ ./configure
- $ sudo make
- $ sudo make install

3. Windows
- www.nodejs.org --> Download the installer --> Install --> set PATH

## Check
- $ node -v
- $ node --> console.log('Hello World');
