# Express Framework
- 대소문자를 구분하지 않는다.

$ sudo npm install -g express
$ express --help

## How to create a project
$ express HelloExpress
$ cd HelloExpress && npm install
$ node app


## Sample Project 
$ npm install express@3.*
$ node_modules/express/bin/express HelloExpress
$ cd HelloExpress && npm install
$ node app

## 9.3.3 실행 환경 설정
$ NODE_ENV=production node app
